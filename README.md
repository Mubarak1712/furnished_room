# 🏡 Furnished Room Visualizer

## 📌 Project Description

The **Furnished Room Visualizer** is a simple web application that allows users to design and visualize a room by selecting different furniture items such as beds, chairs, and lamps.

It helps users get an idea of how a room looks with different furniture arrangements.

---

## 🚀 Features

* 🛏️ Add beds to the room
* 🪑 Add chairs
* 💡 Add lamps
* 🎨 Simple and clean user interface
* ⚡ Fast and lightweight (no backend required)

## ▶️ How to Run the Project

### Method 1 (Simple)

1. Open the project folder
2. Double-click on `index.html`
3. It will open in your browser

### Method 2 (Recommended - VS Code)

1. Open project in VS Code
2. Install **Live Server** extension
3. Right-click on `index.html`
4. Click **Open with Live Server**
